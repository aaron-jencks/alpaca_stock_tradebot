Type any R code in the chunk, for example:

stocks = read.csv('./history.csv') %>%
  group_by(Name) %>%
  mutate(Seconds_Epoch=Second+(Minute*60)+(Hour*3600)+(Day*86400)+(Year*31536000)) %>%
  arrange(Seconds_Epoch) # %>%
  # mutate(dT=)
stocks %>% ggplot(aes(x=Seconds_Epoch, y=Bid_Price)) + facet_wrap(~Name, scales='free') + geom_point() + geom_smooth() + labs(x='Time (sec)', y='Bid Price')
## `geom_smooth()` using method = 'loess' and formula 'y ~ x'

Now, click the Run button on the chunk toolbar to execute the chunk code. The result should be placed under the chunk. Click the Knit and Open Document to built and preview an output.
